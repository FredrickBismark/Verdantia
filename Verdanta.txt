# VERDANTA — Implementation Framework
# Garden Management Application with LLM-Curated Plant Intelligence

## PROJECT IDENTITY

- **Name:** Verdanta
- **License:** MIT
- **Language:** Python 3.11+ (backend), TypeScript (frontend)
- **Target Platform:** Linux Mint on Minisforum UM760 (Ryzen 7 6800H, 16-32GB RAM), but should run on any OS
- **Deployment:** Local-first, single-user, no cloud dependency required
- **Repository Structure:** Monorepo

---

## CORE PHILOSOPHY

1. Everything runs locally by default. Cloud APIs (weather, LLM) are optional enhancers with offline fallbacks.
2. The LLM is not a chatbot bolted on — it is the intelligence layer that curates plant data, generates schedules, and proactively advises.
3. Data transparency: users can always see what sources informed a recommendation and what the LLM was given as context.
4. No paywall, no subscription, no telemetry, no cloud accounts required.
